Smart Waste Segregation System

An AI-powered waste classification web application that helps users identify and segregate waste into categories such as biodegradable, recyclable, and hazardous waste using Machine Learning.

Features
- Upload waste images for prediction
- AI-based waste classification
- Simple and responsive web interface
- Flask-powered backend
- Machine Learning model integration
- Fast and accurate predictions

Technologies Used
- Python
- Flask
- HTML
- CSS
- Machine Learning
- TensorFlow / Keras
- Git & GitHub

Project Structure

├── static/                # CSS, images, static files
├── templates/             # HTML templates
├── app.py                 # Main Flask application
├── train.py               # Model training script
├── requirements.txt       # Dependencies
└── README.md              # Project documentation
